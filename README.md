# Portal de Solicitações API 🚀

API REST desenvolvida com FastAPI para gerenciamento de solicitações internas.

## Funcionalidades
- Criar solicitações
- Listar solicitações
- Buscar por ID
- Atualizar solicitações
- Remover solicitações

## Tecnologias
- Python
- FastAPI
- Uvicorn
- Pydantic

## Executar o projeto

```bash
uvicorn app.main:app --reload
```
